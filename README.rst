=============
 vim-mythryl
=============

Introduction
------------

``vim-mythryl`` provides:
- Mythryl syntax highlighting for ``*.pkg``, ``*.api`` and ``*.my`` files
- Mythryl's ``make6`` syntax highlighting for ``*.make6`` files
